# Yantra-Yukti — welcome / landing page

Public landing page for **Yantra-Yukti** (*"build how things work"*), a private
systems-thinking sandbox where curious kids reverse-engineer everyday
machines and program their hidden logic — and, in a second **Circuits** mode, fix electronic
circuits taught through a water metaphor.

- **Landing page:** served from GitHub Pages (this repo).
- **The app itself:** https://yantra-yukti.web.app (sign in with Google to start).

Self-contained single `index.html` (no build step). Links visitors to the app and to
`support.yantrayukti@gmail.com` for questions. The app's source lives in a **separate**
private repository — this repo is only the public welcome page.
